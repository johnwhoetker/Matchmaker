Application Overview

MatchMate is a simple web-based system designed to match people based on shared interests. Users can create a basic profile, enter their top interests, and the system provides potential matches. The goal is to make connecting with others easier and more fun.

Technology Overview

  • HTML / CSS for the layout
  • JavaScript for simple interactions
  • Hosted using GitHub Pages

Feature List (Prioritized)

  • Create a profile
  • Enter interests
  • Basic matching system

Future Features

  • Chat system
  • AI-powered matching
